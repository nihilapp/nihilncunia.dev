# UI 개발 체크 리스트

본 문서는 블로그 프로젝트의 UI 구현 항목을 자원(도메인)별로 분류하여 정리한 체크리스트입니다.

## 1. 블로그(Blog) UI - 임시

- [ ] 블로그 목록 페이지 (`/admin/blogs`)
- [ ] 블로그 생성 페이지 (`/admin/blogs/new`)
- [ ] 블로그 수정 페이지 (`/admin/blogs/[id]/edit`)
- [ ] 블로그 상세 보기 모달/페이지 (`/admin/blogs/[id]` 또는 모달)
- [ ] 블로그 선택 드롭다운 (내부 컴포넌트)
- [ ] 블로그 상태 변경 UI (활성화/비활성화, 내부 컴포넌트)
- [ ] 블로그 공개/비공개 토글 (내부 컴포넌트)
- [ ] 블로그 통계 대시보드 (`/admin/blogs/[id]/dashboard`)
- [ ] 블로그 설정 페이지 (`/admin/blogs/[id]/settings`)
- [ ] 블로그 테마 선택 UI (내부 컴포넌트)
- [ ] 블로그 카드 컴포넌트 (내부 컴포넌트)

- [ ] 초대 수락/거부 UI (내부 컴포넌트)
