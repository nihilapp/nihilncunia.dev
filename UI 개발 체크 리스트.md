- [ ] 포스트 검색 컴포넌트 (내부 컴포넌트)

- [ ] 조회수/좋아요 표시 UI (내부 컴포넌트)
- [ ] 임시 저장 목록 UI (`/admin/posts/drafts`)
- [ ] 자동 저장 상태 표시 (내부 컴포넌트)
- [ ] 일괄 작업 UI (체크박스, 툴바, `/admin/posts`)

- [ ] 포스트 카드 컴포넌트 (공개/관리자 공통)
