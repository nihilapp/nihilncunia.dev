# 개인 블로그 프로젝트 구상

## 1. 프로젝트 개요
- **목적**: 개인용 블로그 웹사이트 구축
- **기술 스택**: Next.js, React, TypeScript, Prisma, TailwindCSS, shadcn/ui
- **핵심 기능**: 웹 기반 블로그 포스팅 및 관리 시스템

## 2. 콘텐츠 관리 시스템
### 2.1 글 작성 및 저장
- 웹 인터페이스를 통한 마크다운 기반 글 작성
- 모든 글은 데이터베이스에 저장 후 동적 렌더링
- 작성된 마크다운 → DB 저장 → 프론트엔드 렌더링 프로세스

### 2.2 글 작성 페이지 접근 방법
- 어드민 페이지를 통한 직접 접근
- 로그인 상태에서 모든 페이지에 공통 노출되는 관리 UI를 통한 접근

## 3. 인증 및 보안
### 3.1 로그인 시스템
- 관리자 전용 인증 시스템 구현
- 일반 사용자 회원가입 기능 비활성화 (관리자만 접근 가능)

### 3.2 어드민 페이지 보안
- 어드민 관련 페이지는 외부 노출 완전 차단
- robots.txt를 통한 검색 엔진 크롤링 방지
- 사이트맵에서 어드민 페이지 제외

## 4. 카테고리 및 태깅 시스템
### 4.1 카테고리 관리
- 모든 블로그 글에 카테고리 필수 할당
- 웹 인터페이스를 통한 카테고리 생성/수정/삭제 기능

### 4.2 해시태그 시스템
- 글당 최대 10개의 해시태그 설정 가능
- SEO 최적화를 위한 메타태그 활용

## 5. SEO 최적화
### 5.1 검색 엔진 최적화
- 모든 블로그 페이지 SEO 친화적 구조 설계
- 메타 태그, 구조화된 데이터, 사이트맵 등 SEO 요소 완비
- 성능 최적화를 통한 검색 순위 향상

### 5.2 품질 관리
- 정기적인 SEO 점검 및 개선 사항 반영
- 웹 접근성 및 사용자 경험 최적화

## 6. 기술적 요구사항
- **프론트엔드**: Next.js 14+ (App Router), React 18+, TypeScript
- **스타일링**: TailwindCSS + Styled Components + shadcn/ui
- **UI 컴포넌트**: shadcn/ui를 활용한 일관된 디자인 시스템 구축
- **데이터베이스**: Prisma ORM 활용
- **배포**: Vercel 또는 유사 플랫폼
- **패키지 관리**: pnpm 사용
