# 리액트 커스텀 훅 체크 리스트 (API 매칭)

본 문서는 블로그 프로젝트의 React Query 기반 커스텀 훅 구현 항목을 API와 1:1로 매칭하여 정리한 체크리스트입니다.

---

## 📌 기존 API 대응 훅 (단일 블로그 시스템)

### 1. 인증(Auth) 훅

- [ ] **useSignUp** - `POST /api/auth/sign-up` - 새로운 관리자 생성

> **참고**: NextAuth 사용으로 다음 기능들은 NextAuth 내장 함수 사용:
>
> - 로그인: `signIn()` 함수
> - 로그아웃: `signOut()` 함수
> - 세션 조회: `useSession()` 또는 `auth()` 함수

## 📝 요약

- **총 API 엔드포인트**: 약 120개
- **총 커스텀 훅**: 약 120개 (1:1 매칭)
- **기존 시스템 훅**: 약 70개
- **블로그 매니지먼트 시스템 추가 훅**: 약 50개

각 훅은 해당하는 API 엔드포인트와 정확히 매칭되며, React Query의 `useQuery` (GET 요청) 또는 `useMutation` (POST/PUT/PATCH/DELETE 요청) 패턴을 따라 구현됩니다.
